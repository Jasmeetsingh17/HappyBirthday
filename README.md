# Happy Birthday Website

A simple birthday greeting webpage with music, animations, and a fun celebration UI.

## 🚀 Live Demo
This project is hosted on GitHub Pages:

https://jasmeetsingh17.github.io/HappyBirthday/

## 📁 Project Structure
- `index.html` — main landing page
- `last.html` — alternate page
- `style.css` — site styles
- `script.js` — interactive behavior
- `cause.html`, `cause.css`, `cause.js` — extra page content
- `gif1.gif`, `gif2.gif`, `music.mp3` — media assets

## ▶️ How to Run Locally
1. Clone the repo:
   ```bash
   git clone https://github.com/Jasmeetsingh17/HappyBirthday.git
   ```
2. Open `HappyBirthday/index.html` in your browser (or use VS Code Live Server).

## 📌 Notes
- GitHub Pages publishes from the branch root (with `index.html` in the project folder).
- If you update files, commit and push: `git add . && git commit -m "update site" && git push`.
